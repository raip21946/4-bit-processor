# 4-bit-processor
Design and Simulation of a 4-bit Processor using Verilog HDL
Design and Simulation of 4-bit Processor
Overview

This project implements a simple 4-bit processor in Verilog HDL. The processor consists of a Program Counter, Instruction Memory, Control Unit, Register File, ALU, and Data Memory.

Features
4-bit datapath
Custom instruction set
Arithmetic operations:
ADD
SUB
Logical operations:
AND
OR
XOR
NOT
Program Counter
Register File with four registers
Data Memory
Fetch → Decode → Execute architecture
Behavioral simulation using Xilinx Vivado
Tools Used
Verilog HDL
Xilinx Vivado
XSim Simulator
